**Advisor:** [Prof. Ming-Hsuan Yang](https://faculty.ucmerced.edu/mhyang/)

Proposed a spatiotemporal generative network which learns dynamic textures from a single video clip. Demonstrated the proposed algorithm performs favorably against state-of-the-art methods. Designed an encoder that allows the unconditional model to transform an input frame into a video sequence.
